# Job-Tracker (Setup-Start)

## Environment

1. Copy `fake.env` to `.env`
2. Add your real `GEMINI_API_KEY`

## Lokale Ordnerstruktur

- `storage/job-postings/` fuer gespeicherte Stellenanzeigen (z. B. Rohtext oder JSON)
- `storage/applications/` fuer deine Bewerbungsdateien (PDFs)

Diese Ordner sind lokal und werden per `.gitignore` nicht versioniert.
